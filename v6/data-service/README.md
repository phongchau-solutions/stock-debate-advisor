# Data Service

FastAPI microservice for financial data retrieval, caching, and storage.

## Features

- Yahoo Finance API integration for comprehensive financial data
- News crawling from Vietnamese financial sources
- PostgreSQL & MongoDB storage
- Apache Airflow DAG orchestration
- RESTful API endpoints
- Data caching and batch processing

## Running

```bash
docker compose up data-service
```

## Environment

See `.env.example` for required environment variables.
