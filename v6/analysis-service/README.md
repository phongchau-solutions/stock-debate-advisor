# Analysis Service

FastAPI microservice providing multi-methodology financial analysis.

## Features

- Fundamental Analysis (financial ratios, valuation metrics)
- Technical Analysis (trends, indicators, patterns)
- Sentiment Analysis (news sentiment scoring)
- Scoring system (0-100 scale)
- ETL pipeline for data processing
- RESTful API endpoints

## Running

```bash
docker compose up analysis-service
```

## Environment

See `.env.example` for required environment variables.
